# Nockta architecture notes — Shopify Headless

The headless/Hydrogen scaffolder (still provisional upstream tooling) owns
this project's structure (`app/routes/`, `app/components/`, GraphQL codegen
output, caching config). Because the upstream command itself isn't settled
yet, the Nockta `standard` overlay for `shopify-headless` is deliberately
minimal and disturbs nothing generated by the scaffolder.

This overlay only adds this `docs/nockta/` folder for Nockta-specific
project notes that don't belong inside the scaffolder-owned tree —
Storefront API/cart conventions, caching notes, deployment checklists, etc.
